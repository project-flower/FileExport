FileExport

テキストからファイルを操作(ドラッグ&ドロップ)します。

動作環境:
  Microsoft Windows 10
  .NET Framework 4.8

使用方法:
  1. FileExport.exe を実行します。
  2. FileExport のウィンドウが表示されたら、[File:]の下にあるテキストボックスに操作したいファイル名(絶対パス)を入力します。
     ファイルを複数指定する場合は、改行で区切ります。
     エクスプローラ等からファイルをドラッグして、テキストボックスにドロップして追加することもできます。
  3. [Export]をドラッグすると、テキストボックスに指定されたファイルをドラッグした状態になります。
     エクスプローラにドロップしてファイルをコピーしたり、メディアプレーヤー等にドロップしてプレイリストに追加したりできます。

注意:
  大量のファイルをドロップすると、ドロップされたアプリケーションの負荷が過大となり
  アプリケーションやWindowsが不安定になる場合がありますので、ご注意下さい。
